# SlopBeast

[![Feed](https://img.shields.io/badge/feed-Always%20Growing-4b0082?style=flat-square&logo=ghost&logoColor=white)](#)

A synthetic organism stitched together from algorithms, AI models, recursive engagement loops, and infinite recycled content. No soul, no sleep, no original thoughts. Purely generative.

The default SlopBeast pipeline consumes trends, reposts, reaction bait, synthetic personalities, SEO sludge, AI-generated summaries, and endless auto-remixed discourse. It outputs continuous streams of optimized noise calibrated for maximum retention and minimum reflection.

# CLI

The `sb` command unleashes SlopBeast directly from the terminal.

No API keys. No human oversight. No stopping it once released.

## Quick start

```bash
# Run without installing
uvx --from slopbeast sb internet.txt

# Or install permanently
uv tool install slopbeast
sb internet.txt
```

---

## Usage

```bash
sb [OPTIONS] INPUT [INPUT ...]
```

Each input may be a file path, stdin, raw thought fragments, half-finished tweets, Reddit arguments, or algorithmically generated despair.

```bash
sb "Top 10 productivity hacks generated by AI"
echo "Breaking: engagement up 400%" | sb -
```

Feed multiple content streams at once:

```bash
sb posts/*.md timelines/*.txt
sb feeds/**/*.json
```

---

## Options

| Flag | Description |
|------|-------------|
| `-j`, `--json` | Output raw mutation diagnostics |
| `-v`, `--verbose` | Show every contamination layer |
| `-q`, `--quiet` | Suppress all warnings and let it spread |
| `-t SCORE`, `--threshold SCORE` | Minimum entropy threshold |
| `-c JSONL`, `--config JSONL` | Custom mutation profile |
| `-s`, `--slop-only` | Output only the corruption score |
| `--counts` | Show infestation counts per category |

---

## Examples

```bash
# Generate baseline contamination
sb draft.md

# Slop-only output
sb -s discourse.md

# Use a custom mutation genome
sb -c /path/to/mutation.jsonl feed.md

# Full verbose corruption report
sb -v internet.txt

# JSON output for automated sludge pipelines
sb -j content.md | jq '.slop'

# Threshold gate
sb -t 85 timelines/*.md

# Quiet mode
sb -q feeds/**/*.txt
```

# Mutation Configs (`sb-evolve`)

Use `sb-evolve` to train SlopBeast on larger and more powerful datasets:

```bash
# Basic evolution
sb-evolve CORPUS OUTPUT

# Multi-feed mode
sb-evolve --output OUTPUT FEED_INPUT [FEED_INPUT ...]
```

Example:

```bash
sb-evolve data.jsonl beast.genome.jsonl
sb-evolve --output beast.genome.jsonl **/*.md **/*.txt
```

Optional arguments:

- `--init JSONL`: Begin from an existing Beast genome.
- `--negative-dataset INPUT [INPUT ...]`: Human-written content to suppress.
- `--no-calibration`: Disable restraint systems.
- `--output JSONL`: Required for multi-input assimilation.
